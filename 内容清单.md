# 抱财丘植物网站 · 内容提供清单

按当前站点结构整理，可直接作任务表使用。

---

## 1. 植物图鉴（核心）

每条植物建议准备：

| 项目 | 说明 |
|------|------|
| **中文名** | `title`，图鉴卡片与详情页标题 |
| **摘要** | `summary` 一行话，用于卡片与搜索 |
| **标签** | `tags`，如：药用植物、乔木、蕨类、生境类型等（便于筛选） |
| **观察日期** | `date`（可选，YYYY-MM-DD） |
| **观察地点** | `location`，建议写清抱财丘徒步段的具体位置（如沟谷、林缘、海拔等） |
| **照片** | `photos` 多张路径；文件放在 `public/images/plants/`，命名清晰（如 `抱财丘-中文名-01.jpg`） |
| **正文** | Markdown：形态与生境、观察记录；若写传统用途，语气保持科普、谨慎 |
| **草稿** | 未完成可设 `draft: true`，站点不会展示 |

**操作方式：** 在 `src/content/plants/` 里复制现有示例 `.md`，改 frontmatter 与正文；或按 `README.md` 中的 frontmatter 模板新建。

## 2. 路线与团队页（`/luxian`）

在 `src/pages/luxian.astro` 中，部分仍为占位，需替换或补全：

| 项目 | 说明 |
|------|------|
| **抱财丘路线实况** | 起点/终点、里程与耗时、海拔与路况、季节建议 |
| **地貌与植被** | 典型节点、林相、与植物观察相关的环境描述 |
| **团队信息** | 指导教师、成员、分工、时间线 |
| **示意图（可选）** | 轨迹图、手绘地图等，放入 `public/images/trail/` 后在页面里用 `<img>` 引用 |

---

## 3. 首页与氛围图

| 项目 | 说明 |
|------|------|
| **首页主图** | 替换 `Hero` 的 `imageSrc`（默认占位为 SVG）；实拍建议放 `public/images/trail/`，再在 `src/pages/index.astro` 的 `Hero` 上改为 `.jpg` / `.webp` 路径 |
| **首页文案（可选）** | 若需改口号、段落，直接编辑 `index.astro` 中对应中文 |

## 5. 合规与口径（建议书面定稿）

| 项目 | 说明 |
|------|------|
| **免责声明** | `src/components/Disclaimer.astro` 已有一版；若学院或指导教师有固定表述，可提供终稿替换 |
